# 每周考勤表生成器

一个用于自动生成每周考勤表的Python脚本，支持批量生成多周考勤表，并提供图形用户界面。

## 功能特点

- **图形界面操作** - 简单易用的GUI界面，无需命令行操作
- **自定义标题** - 支持用户自定义考勤表标题
- **日期范围选择** - 选择开始和结束日期，自动计算对应的周数
- **批量生成** - 支持一次生成多周的考勤表
- **自动分页** - 员工超过一页容量时自动分页，每页补满空白行
- **格式规范** - 参照标准考勤表格式，包含职位、姓名、日期、星期、备注等列
- **周末标识** - 星期六和星期日自动添加绿色背景标识

## 环境要求

- Python 3.6+
- python-docx

## 安装依赖

```bash
pip install python-docx
```

## 使用方法

### 1. 运行程序

```bash
python generate_attendance.py
```

### 2. 在界面中填写信息

1. **标题设置** - 输入考勤表标题（默认为"每周考勤表"）
2. **日期范围** - 选择开始和结束日期（格式：YYYY-MM-DD）
3. **员工信息** - 输入职位和姓名，点击"添加"按钮

### 3. 生成考勤表

点击"生成考勤表"按钮，文件将保存在当前目录下。

## 输出文件命名规则

- 单周：`标题_年份年第X周.docx`
- 多周：`标题_年份年第X-Y周.docx`

## 考勤表格式

| 列名 | 说明 |
|------|------|
| 职位 | 员工职位 |
| 姓名 | 员工姓名（带斜线表头） |
| 日期 | 对应周的日期（蓝色字体） |
| 星期 | 星期一至星期日（蓝色字体，周末绿色背景） |
| 备注 | 备注信息 |

## 表格规格

- 页面：A4横向
- 列宽：2.45cm
- 表头行高：0.84cm
- 数据行高：0.75cm
- 字体：宋体12pt（小四）
- 每页最多13行数据行，不足自动补空白行
